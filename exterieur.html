Découvrir l'extérieur

Nous allons également pouvoir à l'aide de quelques outils sous Linux découvrir le monde extérieur

Tiny Mtr

Le premier outils que nous allons utiliser est un outil de "traceroute" qui và nous permettre de connaitre toutes les étapes (routeurs) parcourus avant d'atteindre notre cible.

Pour installer Tiny Mtr tapez (en root) apt-get install mtr-tiny

Vous pouvez maintenant utiliser la commande mtr www.google.fr ou www.google.fr est la machine que vous voulez atteindre.

Vous verrez alors apparaitre en temps réel la liste des routeurs parcourus.

Dig

Internet se repose en grande partie sur le protocole DNS. C'est ainsi que l'on est capable d'aller sur www.google.fr et non pas 216.58.210.131

Les entrées DNS sont gérées dans un serveur DNS qui peut être interrogé pour avoir de nombreuses informations.

Sous Linux Dig est un outils puissant qui va nous permettre d'interroger les serveurs DNS

dig www.cnss.ma nous donnera des informations sur la machine www du domaine cnss.ma

Une recherche dans google avec site:cnss.ma nous donnera quelques sous domaines de cnss.ma

whois cnss.ma nous donnera des informations sur le domaine cnss.ma

Les sites comme https://www.iplocation.net/ vous permette de découvrir la localisation géographique d'une adresse IP
